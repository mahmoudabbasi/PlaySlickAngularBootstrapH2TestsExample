Get Started

Main goal of this Example is to show how to test Play Framework in memory using H2 and Slick and keep MySQL as main database.
Also contain one-page application RequireJS + AngularJS with Bootstrap, MySQL and Slick.

For more information please read README at authors Github: https://github.com/pvoznenko/play-slick-angular-test-example/blob/master/README.md

About base template AngularJS + RequreJS you can read at Marius Soutier's blog
